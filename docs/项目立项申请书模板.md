**一、 项目背景与目标**

- **背景**: 简述在数字化竞争中，实时感知竞品动态对企业战略决策的重要性。
- **目标**: 构建一个自动化竞品情报分析平台，能够从公开舆情中自动提取竞品动态，并与本企业数据进行对比，输出结构化的竞争态势简报。

**二、 监控维度与竞品列表**

- **监控维度**:
  1. **价格战**: 监控竞品价格变动、促销活动。
  2. **新品发布**: 监控竞品新产品上线、功能迭代。
  3. **负面舆情**: 监控竞品相关的用户投诉、负面媒体报道。

**三、 技术选型与架构**

- **大模型**: Anthropic Claude (Haiku/Sonnet/Opus)
- **开发框架**: LangChain (Chains, Agents, RAG, Memory)
- **后端服务**: FastAPI
- **向量数据库**: Chroma / FAISS
